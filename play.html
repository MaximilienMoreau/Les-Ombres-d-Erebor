Mode en ligne

Jouer avec vos amis

Connectez-vous via un code de salle. Chaque joueur voit son rôle secret sur son appareil pendant que vous discutez en appel vidéo.

🎥 Compatible Discord / Zoom 📱 Fonctionne sur mobile 8–20 joueurs
⚙️

Configuration requise pour le mode en ligne

Pour synchroniser les parties en temps réel entre appareils distants, configurez un projet Firebase gratuit :

  • Créez un projet sur console.firebase.google.com
  • Activez Realtime Database en mode test
  • Copiez votre config dans assets/js/config/firebase-config.js
  • Passez configured: true dans le fichier

Sans Firebase, utilisez le mode local (toutes les cartes sur un seul appareil) décrit plus bas.

🏰

Créer une salle

Vous serez le Maître du Jeu. Vous configurez les rôles, gérez les phases et arbitrez la partie. Les autres joueurs vous rejoignent avec le code que vous recevrez.

🚪

Rejoindre une salle

Entrez le code de salle partagé par le Maître du Jeu. Vous rejoindrez le salon d'attente et recevrez votre rôle secret quand la partie commencera.

Fonctionnement

Comment jouer en ligne

Le site gère les rôles et les votes. Vous gérez la discussion en appel vidéo.

🏰

1. Créer la salle

Le MJ crée la salle et reçoit un code à 6 caractères.

📤

2. Partager le code

Le MJ envoie le code via Discord, WhatsApp, etc.

🃏

3. Attribution des rôles

Le MJ assigne les rôles. Chacun voit le sien sur son écran.

🌑

4. La nuit tombe

Les joueurs soumettent leurs actions nocturnes sur le site.

☀️

5. Discussion & Vote

Débattez en appel vidéo, puis votez sur le site.

Alternative

Mode Local — Même réseau

Jouez sans configuration si tous les joueurs sont au même endroit.

Aucun compte requis

Si tous vos joueurs sont dans la même pièce, utilisez le mode local. Un seul appareil fait office de Grimoire pour le MJ. Les cartes de rôle sont partagées individuellement (l'appareil passe de main en main au démarrage).

Parfait pour les soirées de jeu où tout le monde est présent !

🎮 Lancer en mode local
📱Un seul appareil pour le MJ (Grimoire)
🔐Rôles affichés en privé à chaque joueur
🌐Pas de connexion internet requise
⚡Démarrage immédiat, aucune configuration
